# 📊 Netflix Data Analysis Dashboard (Tableau)

🚀 A data visualization project showcasing business insights using Tableau.

---

## 🧠 Project Overview

This project analyzes the Netflix Movies & TV Shows dataset using Tableau to uncover meaningful insights about content distribution, trends, and audience preferences.

The goal of this project is to:

* Understand how Netflix content has grown over time
* Compare Movies vs TV Shows
* Identify popular genres and ratings
* Analyze country-wise content production

---

## 🎯 Objectives

* Analyze Netflix content trends over the years
* Compare distribution of Movies and TV Shows
* Build an interactive and visually appealing dashboard

---

## 📂 Dataset

* Source: Netflix Dataset (GitHub)
* Total Records: ~7000+

### Columns Used:

* Title
* Type (Movie / TV Show)
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre (Listed In)
* Description

---

## 🛠️ Tools & Technologies

* Tableau (Data Visualization)
* Microsoft Excel (Data Cleaning & Preprocessing)

---

## 📊 Dashboard Features

* 📈 Content added over the years
* 🎬 Movies vs TV Shows distribution
* 🌍 Country-wise content analysis
* ⭐ Rating distribution
* 🔎 Interactive filters (Year, Country, Type)

---

## 💡 Key Insights

* Netflix content significantly increased after 2015
* Movies are more in number compared to TV Shows
* The United States produces the highest content
* Most content falls under the TV-MA rating category
* Drama and International genres are highly dominant

---

## 🚀 How to Use

1. Download the `.twb` file from this repository
2. Open it using Tableau Desktop
3. Connect the dataset if required
4. Explore the dashboard using filters and visuals
